Tech Stack

Backend: Node.js, Express

Database: MongoDB

Frontend: HTML, CSS, JavaScript

Package Manager: npm

No frameworks, no overengineering, no magic.

What This Project Does

As of now, the website includes:

A home page

A recipe section with predefined recipes

A blog system

Add blogs

Remove blogs

A MongoDB-backed admin/block page to manage blog content

A simple character picker feature

Everything is basic, functional, and straightforward.

Getting Started
Prerequisites

Node.js installed

MongoDB installed and running locally

Setup

Clone the repository

git clone https://github.com/BlazeeeXD/PlateUp.git
cd PlateUp


Install dependencies

npm install


Seed the database
MongoDB credentials are not included.
JSON seed files are provided in the repository ‚Äî import them into your local MongoDB instance before running the app.

Start the server

npm start


Open the app in your browser
(Usually at http://localhost:3000, depending on your setup.)
